What Is This?

This is a tool for reading International Phonetic Alphabet (IPA) notations and predicting their American English translation.

It uses IBM Watson and Google Cloud API to predict English word(s) based on the pronounciation of word(s) with IPA characters.

For example, "budgerigar" (AKA the common parakeet) can be pronounced according to its IPA notation /ˈbʌdʒərɪɡɑːr/. If you type ˈbʌdʒərɪɡɑːr, tool will predict the English word "budgerigar".

How Do You Use It?

Find an IPA word you would like an English translation of. Type it in, without the slashes, in the left text box and you will see the closest English prediction in the right text box. IPA words can often be found in Wiktionary.

For example, ˈpɪdʒən will be read as "pigeon."

Why did I build this?

The inspiration to build such a tool came from this post.

I was writing matrix comparison algorithms to match predicted phonemic values of different IPA characters for the Synthetic Telepathy Project for NeuroTechSC which increased my curiousity about IPA letters and I came across this post that gave me an idea to build a simple tool. I also wanted to get into web development so I used this project to learn using React JS and Django.

This tool will go through many rapid changes, particularly in the beautification of its UI and the addition of some more features, as I gain more experience with web development.
